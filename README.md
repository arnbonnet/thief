# Thief

### Le but du jeu est de voler le plus d'objets possible dans le manoir et de s'enfuir.

#### Au départ vous avez :
* 3 points de vie
* 1 outil
* Un niveau d'alerte initialisé à 0

#### Vous aurez perdu si :
* Vos points de vie atteignent 0
* Le niveau d'alerte atteind 4
* Dans d'autres cas secrets ...
